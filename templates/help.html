常见问题 FAQ

  1. 进入"任务管理"页面
  2. 点击"创建任务"按钮
  3. 选择任务类型（如内容采集、视频剪辑等）
  4. 填写任务参数和配置
  5. 设置执行时间和优先级
  6. 点击"保存"创建任务

  1. 进入"账号管理"页面
  2. 点击"添加账号"
  3. 选择平台类型（抖音、B站、小红书等）
  4. 输入账号信息和认证凭据
  5. 测试连接确保配置正确
  6. 保存账号配置

  1. 检查系统监控页面，查看CPU、内存使用情况
  2. 清理临时文件和日志文件
  3. 减少同时运行的任务数量
  4. 优化任务参数和配置
  5. 考虑升级硬件配置

备份：

  1. 进入系统配置页面
  2. 点击"数据备份"
  3. 选择备份内容（数据库、配置文件、媒体文件）
  4. 下载备份文件

恢复：

  1. 停止系统服务
  2. 上传备份文件
  3. 执行恢复操作
  4. 重启系统服务

  1. 进入"系统日志"页面
  2. 选择要查看的日志文件
  3. 使用过滤器筛选特定级别或时间范围的日志
  4. 使用搜索功能查找特定关键词
  5. 可以下载日志文件进行离线分析
